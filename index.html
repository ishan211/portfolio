IshanOS

Created using: x86-16 Assembly

A fully functional operating system made in x86-16 bit Assembly to run on a [x]. Featuring games, a text editor, a FAT12 file system, and a simple BASIC-style interpreted programming language. IshanOS is less than 1.44MB and can fit entirely on a single 3.5" Floppy Diskette.
